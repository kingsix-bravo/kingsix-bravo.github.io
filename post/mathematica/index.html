追答
“->”是Mathematica里最重要指令之一Rule的简写……要“去掉”的话，可以：
yy /. List/@{yy -> 8.90323265887200425958769930845,
yy -> 8.90323265943529241591021289811,
yy -> 8.90323265800195651367281533850}

道理：

In[111]:= x /. x -> 3
Out[111]= 3

In[112]:= x /. {x -> 3}
Out[112]= 3

In[81]:= x /. {{x -> 3}}

Out[81]= {3}

In[113]:= x /. {{{x -> 3}}}
Out[113]= {{3}}
